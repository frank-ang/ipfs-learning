# IPFS notes.

```sh
# Run daemon
ipfs daemon

# Show node stats
ipfs repo stat
ipfs stat repo 


# default peers to learn about other peers
ipfs bootstrap list

# List directories in the local mutable namespace.
ipfs ls [<path>]

# Display file status.
ipfs files stat <path>

# Add file
ipfs add [--recursive] <path>
```
